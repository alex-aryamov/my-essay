1. Для чего нужен файл README.md в репозитории?
 Ответ: README.md - нужен для описания проекта и то как им пользоваться, для написания использованных источников

2. Какую информацию обычно размещают в файле README.md?
  Ответ: Файл README.md нужен для:
    -Описание проекта
    -Инструкции по установке
    -Примеры использования
    -Информацию о лицензии

3. Как создать заголовок первого уровня в Markdown?
  Ответ: Поставить перед текстом заголовка знак #

4. Как создать нумерованный список в Markdown?
  Ответ: поставить цыфру с точкой перед текстом, а для вложенного нумерованного списка нужно нажать сначала пробел и уже потом цыфру с точкой
  Пример:
  1. Первый пункт
  2. Второй пункт
    1. Вложенный пункт

5. Для чего нужна вики-страница в репозитории?
  Ответ: Wiki - нужна для документации с элементами :
    -Руководствами
    -FAQ
    -Примеры кода
    -Описанием API
